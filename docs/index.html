Act versus Actor

How criminal courts describe sexual violence differently depending on the victim’s gender.

Female-victim opinions name the violent act more often. Male-victim opinions name the perpetrator more often.

97 U.S. judicial opinions · balanced by gender · stratified by procedure · features computed on opinion text only (editorial headnotes excluded).

The question

When a court writes about sexual assault, the facts can be the same—but is the language? This project asks how victim gender shapes the way U.S. judicial opinions frame violence and the people involved: when does the record foreground the act, and when the actor?

The data

97 U.S. sexual-assault opinions retrieved from CourtListener, balanced by victim gender. Victim gender is entangled with case type—women dominate the criminal cases, men the civil ones—so every comparison is run within a stratum, never pooled.

Male victims overwhelmingly access civil suits; female victims, criminal prosecution—only 34% of male-victim cases are criminal, versus 68% of female-victim cases.

The method

Each opinion’s text (headnotes stripped) is run through an NLP pipeline producing 140+ per-case measures, including 19 framing detectors, as per-sentence rates. Male- and female-victim cases are compared within each stratum using effect sizes, not just p-values.

1

Pull

Retrieve opinions from CourtListener; keep opinion text only.

2

Features

Extract 140+ measures, incl. 19 framing detectors, as per-sentence rates.

3

Compare

Male vs. female within each stratum (criminal / civil).

4

Test

Cliff’s δ, 500× bootstrap CIs, Benjamini–Hochberg FDR + Bonferroni.

The findings

Within criminal cases, the same crime is written with opposite grammar depending on the victim’s gender.

δ = −0.62
Naming the act — female-victim opinions name the violence directly far more often.
δ = +0.63
Naming the actor — male-victim opinions name the perpetrator far more often.

Female-victim opinions name the violence more directly—and the gap is largest within criminal cases.

Male-victim opinions name the perpetrator more often—largest within criminal cases, and gone within civil cases.

Across every framing detector, only two clear strict correction: naming the violence and naming the perpetrator.

Why it matters

The legal record centers the act for women, and the actor for men.

For female victims the violence is spelled out; for male victims attention shifts to the person who did it, and the harm itself recedes. This is a concrete, measurable linguistic mechanism behind the male-victim invisibility the literature describes but rarely quantifies—and it holds on opinion-only text, within case type, surviving the strictest corrections.
